## 🛠️ Global Core Competencies

Across these projects, the following industry-standard practices are implemented:

* **Infrastructure as Code (IaC):** Extensive use of Terraform for modular, reusable cloud provisioning (VPCs, EKS clusters, S3 buckets).

* **Containerization & Orchestration:** Deep integration with Docker for image building and Kubernetes (EKS) for high-availability scheduling.

* **CI/CD Automation:** Robust pipelines defined in Jenkinsfiles for automated testing, building, and rolling updates to production.

* **Cloud Security & Management:** Implementation of IAM OIDC providers, Security Groups, and S3 Remote Backends for secure state management.
